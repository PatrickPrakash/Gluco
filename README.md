# Gluco
Open Source Diabetes Management

Gluco is a Open -Source Diabetes Management System.

Features:

* Cloud-Based

* Material Design

* Track Blood glucose level,Blood Pressure,Body Weight

* Get a Detailed graph overview of your diabetes.

* Track your Activity

* Get Diet Tips
